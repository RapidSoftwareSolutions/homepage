Intigrating with iOS

This tutorial is for developers building iOS apps using the swift programing language.

 

To make your app interact with your RapidAPI system, all you have to do is make an HTTP request to an endpoint in your project. While you can do this with the built in networking libraries such as NSURLConnection, it’s usually more comfortable to use one of the many widely available open source libraries, as they generally make life much easier and automate most of the process for you.

 

The library we will be using is called SwiftHTTP, and it’s very popular and simple to use in Swift applications.

 

The first step will be to install SwiftHTTP in your project which can be done natively / using cocoapods.

 

Installing with Cocoapods:

If you have never used cocoapods, check out Get Started tab on cocoapods.org.

 

In your Podfile just add

 

source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'
use_frameworks!
pod 'SwiftHTTP', '~> 0.9.2'

And then run the following line in the terminal in the project directory

        

        pod install

 

 

Installing manually:

Simply grab the framework (can be downloaded from github:https://github.com/daltoniam/SwiftHTTP) and add the SwiftHTTP.xcodeproj to your Xcode project. Once that is complete, in your "Build Phases" add the SwiftHTTP.framework to your "Link Binary with Libraries" phase.

 

Importing SwiftHTTP:

Now that you have SwiftHTTP installed in your project, import it where you want to use it by adding

import SwiftHTTP


Paths:

To access a certain endpoint in your project, you’ll have to use it’s URL. When you created the endpoint, you gave it a certain path (something like /blahblah). So the URL you will use to access it is:

https://yourProjectName.imrapid.io/blahblah

 

Making a GET requests:

If you have a GET endpoint you have created and you want to access it, use the following code, which will get a string of text from that endpoint:

 

var request = HTTPTask()
request.GET("https://projectName.imrapid.io/path", parameters: nil, success: {(response: HTTPResponse) in
       if let data = response.responseObject as? NSData {
           let str = NSString(data: data, encoding: NSUTF8StringEncoding)
           println("response: \(str)") //prints the HTML of the page
       }
   },failure: {(error: NSError, response: HTTPResponse?) in
       println("error: \(error)")
   })

 

If you want to get an object from your endpoint (for example, if the endpoint returns JSON or returns a document it found in a database), use:

 

var request = HTTPTask()
//The parameters will be encoding as JSON data and sent.
request.requestSerializer = JSONRequestSerializer()
//The expected response will be JSON and be converted to an object return by NSJSONSerialization instead of a NSData.
request.responseSerializer = JSONResponseSerializer()
request.GET("https://projectName.imrapid.io/path", parameters: nil, success: {(response: HTTPResponse) in
       if let dict = response.responseObject as? Dictionary<String,AnyObject> {
           println("example of the JSON key: \(dict["key"])")
           println("print the whole response: \(response)")
       }
   },failure: {(error: NSError, response: HTTPResponse?) in
       println("error: \(error)")
   })

 

In this example, dict will be the object received from the endpoint.

 

Making a POST requests:

Making a POST request is a lot like making a GET request:

 

var request = HTTPTask()
//we have to add the explicit type, else the wrong type is inferred. See the vluxe.io article for more info.
let params: Dictionary<String,AnyObject> = ["param": "param1", "array": ["first array element","second","third"], "num": 23, "dict": ["someKey": "someVal"]]
request.POST("https://projectName.imrapid.io/path", parameters: params, success: {(response: HTTPResponse) in

   },failure: {(error: NSError, response: HTTPResponse?) in

   })

 

Notice that using the parameters arguments you can pass the request body arguments.

 

Other request types:

Similarly, you can also make a PUT request:

 

var request = HTTPTask()
request.PUT("https://projectName.imrapid.io/path", parameters: ["param": "hi", "something": "else", "key": "value", success: {(response: HTTPResponse) in
   //do stuff
   },failure: {(error: NSError, response: HTTPResponse?) in
   //error out on stuff
   })

 

And a delete request:

 

var request = HTTPTask()
request.DELETE("https://projectName.imrapid.io/path", parameters: nil, success: {(response: HTTPResponse) in
       println("DELETE was successful!")
   },failure: {(error: NSError, response: HTTPResponse?) in
        println("print the error: \(error)")
   })

 

 

 

And that’s the basic of making HTTP requests to your project from your iOS apps. For more information, you can head over to https://github.com/daltoniam/SwiftHTTP .
