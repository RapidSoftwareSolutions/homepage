Talk with Friends

Building apps with RapidAPI and PubNub is incredibly simple. This tutorial will show you how to quickly build a messaging app in both iOS and Android in under 100 lines of code. Before we start, this tutorial will introduce you to a few technologies we will be using.

RapidAPI

RapidAPI is a system of quickly building a backend without programming. We will be using RapidAPI simply to send messages to others. If this is your first time building something with RapidAPI, check out our some of our other tutorials.

PubNub

PubNub is a service that makes it incredibly simple to create a publish & subscribe system. We will be using it to subscribe to any message sent by our app or anyone else using our app.

Ionic

Ionic is way of quickly building iOS and Android applications simultaneously. The applications are written just as you would build the frontend of a website using AngularJS, HTML, CSS and JavaScript.

Getting Started

For this tutorial, we assume that you using OS X or a Linux distribution. To get started, you will need to install Ionic. To do this, simply open up your terminal and type sudo npm install -g ionic. You will then need to type your password. If you would like to follow along with the finished product, you can now run git clone https://www.github.com/rapidsoftwaresolutions/pubnubchat. Otherwise, start a new project by running ionic start myproject where myproject is the name of your project. However before we start working on the app, we need to setup a project on RapidAPI and get our keys from PubNub.

Getting Keys from Pubnub

To get keys from PubNub, go to pubnub.com and click the getting started button. You will then be asked to make an account. You can use the publish and subscribe keys provided to you for free from PubNub.

Building the App

Now that you have an Ionic app, you will be making changes in myproject/www/js, myproject/www/templates and myproject/www/css.

First, we will create the main Angular app, the app.js file. This will be used for routing between pages. Move this file to exist at the same level as index.html so that it is no longer contained within the js folder. You may now rename the app.js file to index.js.

We need to add our apps dependencies to the module. We will be using ionic, a login module named login, a chat module named chat and the PubNub Angular module. At the top of your index.js replace any previous module declarations with the following:

var app = angular.module('rapidchat', ['ionic','login', 'chat', 'pubnub.angular.service'])

Once this is done, we need to configure the states used by our app. We will have one state for login and one for the actual chat page. To set this up, we write the following code, after the .run function.

.config(function($stateProvider, $urlRouterProvider)
    {
     $stateProvider
      .state('chat', {
        url: "/chat/:nickname",
        templateUrl: "templates/chat.html",
        controller: "chatController"
      })
      .state('login', {
        url: "/login",
        templateUrl: "templates/login.html",
        controller: "LoginController"
     });
       $urlRouterProvider.otherwise('/login');
    })

This sets up both pages in angular, letting our app know where our HTML and JS is located. It also prevents people from going to any other pages that we haven't written, instead redirecting them back to the login page.

Next, add two files in the js folder, login.js and chat.js. For login.js, add the following code. It simply takes the user's nickname, records it, and sends us to the primary page.

The file is relatively short. 

"use strict"
angular.module('login', []).controller('LoginController',['$scope', '$state', function($scope,$state) {
     $scope.login = function(nickname){
        if(nickname)
         {
           $state.go('chat',{nickname:nickname})
          }
       }
}]);

Next, in chat.js we will need to add a way to send and receive messages. Create a new module, named chat in which we will add a controller. This should appears as follows:

"use strict"
    var chat = angular.module('chat', []);
    chat.controller('chatController', ['$scope', '$stateParams', 'PubNub', '$http', function($scope,    $stateParams, PubNub, $http){ }]);

Since we have already set up an endpoint to send messages, we simply need to send a post request to our endpoint. To do this, simply write a post request within our controller. Using Angular, we just write: 

$scope.sendMessage = function(){
    $http.post('https://chattrbox.imrapid.io/sendtext', {message : {nickname: nickname , message : $scope.message}})
    $scope.message = '';
}

Here $scope.messages and $scope.sendMessage let Angular know that these values will be taken from and sent to HTML. This function simply takes the text stored in $scope.message and sends it to our endpoint. Next, we need to include a way of listening for sent messages. With PubNub, this is easy. Simply initialize PubNub and listen for an event by writing the following code:

PubNub.init({
    publish_key: PUBLISH KEY,
    subscribe_key: SUBSCRIBE KEY,
})
PubNub.ngSubscribe({ channel: 'chattrbox' })
$scope.$on(PubNub.ngMsgEv('chattrbox'), function(event, payload) {
    $scope.messages.push(payload.message)
});

Together, the entirety of chat.js should appears as follows:

"use strict"
var chat = angular.module('chat', []);
chat.controller('chatController', ['$scope', '$stateParams', 'PubNub', '$http', function($scope,    $stateParams, PubNub, $http){
const nickname = $stateParams;
$scope.messages = [];
PubNub.init({
    publish_key: PUBLISH KEY,
    subscribe_key: SUBSCRIBE KEY,
})
PubNub.ngSubscribe({ channel: 'chattrbox' })

$scope.sendMessage = function(){
    $http.post('https://chattrbox.imrapid.io/sendtext', {message : {nickname: nickname , message : $scope.message}})
    $scope.message = '';
}

 $scope.$on(PubNub.ngMsgEv('chattrbox'), function(event, payload) {
    $scope.messages.push(payload.message)
})

}]);
chat.directive('ngEnter', function () {
    return function (scope, element, attrs) {
        element.bind("keydown keypress", function (event) {
            if(event.which === 13) {
                scope.$apply(function (){
                    scope.$eval(attrs.ngEnter);
                });
                event.preventDefault();
            }
        });
    };
});
